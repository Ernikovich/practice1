# Moving Ball Game

## 🎯 Objective
Interactive game with a moving red ball.

## Features
- Red ball (radius 25) on white background.
- Moves with arrow keys (Up, Down, Left, Right).
- Each key press moves ball by 20 pixels.
- Ball cannot leave screen boundaries.
- Smooth animation with frame rate control.

## Controls
- Arrow keys to move the ball.

## Run
```bash
python main.py
