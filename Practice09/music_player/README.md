# Music Player

## 🎯 Objective
Interactive music player with keyboard controls.

## Features
- Play, Stop, Next, Previous track functions.
- Displays current track name and playback position.
- Progress bar shows playback progress.
- Playlist management.

## Keyboard Controls
- `P` = Play
- `S` = Stop
- `N` = Next track
- `B` = Previous track
- `Q` = Quit

## Run
```bash
python main.py
