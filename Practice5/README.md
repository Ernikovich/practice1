# 📄 README — Receipt Parser

## Description
`receipt_parser.py` is a Python script for automatically extracting data from a text receipt (`raw.txt`).  
It can extract:

- Product names  
- Prices  
- Total sum  
- Date and time of purchase  
- Payment method  

The results are output in **JSON format** for easy use in other programs.

---

## Requirements
- Python 3.x  
- Built-in `re` module  
- Built-in `json` module  

No external dependencies needed.
