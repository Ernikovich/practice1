# 🎮 Practice 10: Games Snake, Racer, Paint

## 📌 Project Overview
Practice 10 includes three mini‑games written in Python using the **Pygame** library:

- 🐍 **Snake** — the classic Snake game.
- 🚗 **Racer** — an arcade game where the player controls a car and avoids obstacles.
- 🎨 **Paint** — a simple drawing application with multiple tools and colors.

This project helps reinforce skills in:
- Handling keyboard and mouse events,
- Drawing shapes and images,
- Managing the game loop,
- Structuring code with functions.

---

## 🐍 Snake
- Control the snake with arrow keys.
- Goal: collect food to grow longer.
- Collision with walls or itself ends the game.

---

## 🚗 Racer
- Control the car with arrow keys.
- Avoid collisions with other cars and obstacles.
- Speed gradually increases, making the game harder.

---

## 🎨 Paint
- Modes:
  - ✏️ **Pen** — draw freehand lines.
  - 🟦 **Rectangle** — draw rectangles.
  - ⚪ **Circle** — draw circles.
  - 🧽 **Erase** — erase with a white brush.
- Change brush color.
- Clear the screen.
- Display current brush radius in the corner.

---

## ⚙️ Controls
### Snake
- ⬆️⬇️⬅️➡️ — move the snake.

### Racer
- ⬅️➡️ — move the car.
- ⏸️ — pause.

### Paint
- **P** — pen  
- **R** — rectangle  
- **C** — circle  
- **E** — eraser  
- **Q** — clear screen  
- **1–5** — colors (black, green, red, blue, yellow)
